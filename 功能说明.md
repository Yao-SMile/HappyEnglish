# 📱 快乐学英语 - 功能说明

## 应用简介

**快乐学英语** 是一款专为**小学四年级学生**设计的英语单词学习 APP，采用**AI 智能出题**技术，让背单词变得像玩游戏一样有趣！

---

## 🎯 核心功能

### 1️⃣ 单词学习 📚

**学什么？**
- 人教版四年级上册（Unit 1-6）
- 人教版四年级下册（Unit 1-4）
- 共 100+ 核心单词

**怎么学？**
```
┌─────────────────────────┐
│   classroom             │  ← 单词
│   /ˈklɑːsruːm/          │  ← 音标
│   教室                  │  ← 中文意思
│                         │
│   🔊 点击播放发音       │  ← TTS 语音
│                         │
│   This is our classroom.│  ← 英文例句
│   这是我们的教室。      │  ← 例句翻译
└─────────────────────────┘
```

**学习模式：**
- 按单元学习（10 个单元）
- 逐个单词学习
- 自动记录学习进度

### 2️⃣ AI 智能练习 🤖

**这是本 APP 的核心亮点！**

#### AI 如何出题？

```
传统练习：
❌ 固定题目，所有人都做一样的题
❌ 难度不变，不会的永远学不会

AI 智能练习：
✅ 根据你的学习情况动态出题
✅ 优先出你容易错的题
✅ 难度会自动调整
✅ 越学越聪明！
```

#### 出题逻辑

**第一步：选单词**
AI 会优先选择这些单词出题：
- 🔴 你还没学过的
- 🟡 你经常出错的
- 🟠 你好久没复习的
- 🟢 学习次数不够的

**第二步：定难度**
```
单词本身难度 + 你的掌握程度 = 题目难度

例如：
- 简单单词 + 没学过 = 难度 2
- 简单单词 + 已掌握 = 难度 3
- 困难单词 + 没学过 = 难度 3
- 困难单词 + 已掌握 = 难度 4-5
```

**第三步：选题型**
```
难度 1-2（简单）：
→ 选择题（4 选 1）
→ 看图选词

难度 3（中等）：
→ 选择题（5 选 1）
→ 听音选词
→ 填空题

难度 4-5（困难）：
→ 拼写题（默写）
→ 填空题（5-6 选 1）
```

**第四步：自适应调整**
```
答对了 → 下一题难度 +1 ⬆️
答错了 → 下一题难度 -1 ⬇️
```

```

```

```

C. teacher
D. student
```

**④ 看图选词** 🖼️
```

D. garden
```

